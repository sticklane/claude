## Findings

1. **`map` ranking noise on real repos** вЂ” top of `ctx map` on this repo is
   bash test-scratch variables (`hooks.plugin-autorefresh.test.t#1вЂ¦#10`)
   ahead of real API classes/functions. Ranking should down-weight
   variables (especially bash locals / `#N`-deduplicated ones) relative to
   functions, classes, and methods. Filed as draft stub
   `tasks/16-map-ranking-bash-locals.md`.
2. **No Rust extractor** вЂ” ctx cannot index its own `src/` (Rust is not
   among the 12 shipped grammars; a spec-scoping fact, not a defect вЂ”
   recorded here so the next person doesn't rediscover it by surprise).
